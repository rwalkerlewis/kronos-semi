# Benchmark: axisymmetric 2D MOSCAP, LF/HF C–V

Axisymmetric (cylindrical) 2D MOS capacitor solved on the meridian
half-plane $(r, z)$, reproducing Hu Fig. 5-18 (Modern Semiconductor
Devices for IC, ch. 5).

## Device parameters (Hu Fig. 5-18)

- Body: p-Si, $N_a = 5\times10^{16}$ cm$^{-3}$.
- Gate oxide: SiO$_2$, $T_{\text{ox}} = 10$ nm.
- Gate workfunction: $\phi_{ms} = -0.95$ V (n+ poly on p-Si).
- Gate radius $R_g = 5$ µm; outer radial extent $R = 200$ µm
  (≫ 5 $W_{\text{dmax}}$).
- Gate sweep: $V_g \in [-2.0, +2.0]$ V at 0.05 V steps.

## Running locally

```bash
# Regenerate the mesh (optional; .msh is committed)
cd benchmarks/moscap_axisym_2d
gmsh -2 moscap_axisym.geo -o moscap_axisym.msh

# Drive the analytical pipeline (no dolfinx required)
python -c "from semi.cv import analytical_moscap_params; print(analytical_moscap_params())"
```

The dolfinx-gated FEM run is currently driven from the notebook; a
CLI driver is on the follow-up list.
